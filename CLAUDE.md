## Overview
This project implements a currency converter API that reads conversion rates from a CSV file. The API is built with TypeScript and designed to run both locally for development and on Cloudflare workers for production deployment. 

## Technlogies Used
- TypeScript
- Cloudflare Workers
- Wrangler CLI
